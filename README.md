Kickstart creating your own webpack scaffold

## Installation
```bash
$ npm install -g webpack-scaffold-starter
```

## Usage
Create an empty folder, `cd` into it and run following commands
```bash
$ git init # optional
$ webpack-scaffold
```
Have fun creating your own webpack-scaffold !!
